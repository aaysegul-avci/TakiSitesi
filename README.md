# FLORIA Taki Sitesi

Bu proje, modern kullanıcı deneyimi ve estetik arayüz tasarımı hedeflenerek geliştirilmiş bir takı satış sitesi uygulamasıdır.

Proje kapsamında kullanıcıların ürünleri görüntüleyebileceği, ürün detaylarını inceleyebileceği, sepete ürün ekleyebileceği ve temel e-ticaret işlemlerini gerçekleştirebileceği responsive bir web arayüzü geliştirilmiştir.

# Projenin Amacı

Bu projenin amacı, kullanıcı dostu ve modern tasarıma sahip bir takı satış sitesi geliştirmektir. Proje içerisinde temel e-ticaret yapıları oluşturularak responsive bir kullanıcı deneyimi hedeflenmiştir.

---

# Proje Özellikleri

## Kullanıcı İşlemleri

- Ürün listeleme
- Ürün detay sayfaları
- Sepete ürün ekleme
- Kullanıcı giriş ve kayıt ekranları
- Ödeme sayfası görüntüleme
- Responsive tasarım desteği

## Yönetim Özellikleri

- Admin panel yapısı
- Ürün yönetim ekranları
- SQL veritabanı desteği

---

# Kullanılan Teknolojiler

- SQL
